Ubuntu 下 OpenVINO 安装测试

适用于 Ubuntu 24.04，硬件平台 Core i9-14900K + Intel Arc Pro B60；同样适用于 PTL 的 iGPU。页面内所有命令块均可一键复制。

Ubuntu 24.04 OpenVINO / OpenVINO GenAI Intel GPU / Level Zero Conda + Python 3.10
目录 1. 更新系统与内核 2. 安装 Miniconda 3. Conda 换源与环境 4. 安装 Intel GPU 驱动 5. pip 换源 6. 安装 OpenVINO 7. 验证设备与环境 8. 下载与转换模型 9. GenAI 推理测试 10. Benchmark 测试 11. VLM Benchmark

1. 更新系统与升级内核

安装好 Ubuntu 24.04 后先更新源。安装时如果已经选择清华源，可以直接执行：

更新系统
sudo apt update
sudo apt upgrade

升级内核很重要，否则可能识别不到 Level Zero Device。

升级 HWE 内核并重启
sudo apt install linux-generic-hwe-24.04
uname -r
sudo modprobe i915
sudo reboot

2. 下载并安装 Miniconda

可以从官方源下载；如果下载较慢，可改用清华源。

下载 Miniconda
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
wget https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-latest-Linux-x86_64.sh

网络不稳定时，可以使用断点续传：

断点续传下载
wget -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-latest-Linux-x86_64.sh

也可以简单验证源的下载速度：

验证下载速度
wget -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/
安装 Miniconda
bash Miniconda3-latest-Linux-x86_64.sh
source .bashrc

如果不想每次打开终端都自动进入 base 环境：

关闭自动进入 base
conda config --set auto_activate_base false

3. Conda 换国内源并创建环境

配置 Conda 清华源
conda config --remove-key channels
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
conda config --set show_channel_urls yes
conda config --set channel_priority strict
验证 Conda 源
conda config --show channels

建议使用 Python 3.10 创建 OpenVINO 环境。

创建并进入 openvino_env
conda create -n openvino_env python=3.10 -y
conda env list
conda activate openvino_env
执行 conda env list 后，应能看到 openvino_env。

4. 安装 Intel GPU 驱动

先创建安装脚本：

新建脚本文件
nano intel-gpu-installer-no-proxy.sh

将下面内容写入脚本：

intel-gpu-installer-no-proxy.sh
#!/bin/bash

sudo apt-get update
sudo apt-get install -y software-properties-common

sudo add-apt-repository -y ppa:kobuk-team/intel-graphics

sudo apt-get update
sudo apt-get install -y \
    libze-intel-gpu1 libze1 intel-metrics-discovery intel-opencl-icd clinfo intel-gsc

sudo apt-get install -y \
    intel-media-va-driver-non-free libmfx-gen1 libvpl2 libvpl-tools libva-glx2 va-driver-all vainfo

sudo apt-get install -y \
    libze-dev intel-ocloc

sudo apt-get install -y \
    libze-intel-gpu-raytracing xpu-smi

sudo gpasswd -a ${USER} render
newgrp render

sleep 3

echo "Run: clinfo | grep \"Device Name\""
clinfo | grep "Device Name"

运行脚本，然后补充 /dev/dri 相关权限并重启。

运行驱动安装脚本
bash intel-gpu-installer-no-proxy.sh
sudo usermod -aG render $USER
sudo usermod -aG video $USER
sudo reboot

6. 安装 OpenVINO 相关组件

-i 参数表示临时使用国内 PyPI 源。

安装基础包
pip install --upgrade pip setuptools wheel -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install openvino -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install openvino-genai -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install modelscope -i https://pypi.tuna.tsinghua.edu.cn/simple

如果要用 OpenVINO 跑 LLM，还需要安装 optimum 与 transformers。

安装 LLM 依赖
pip install optimum[openvino] transformers -i https://pypi.tuna.tsinghua.edu.cn/simple

完整开发依赖可选安装：

安装开发依赖，可选
pip install openvino-dev -i https://pypi.tuna.tsinghua.edu.cn/simple

ONNX / PyTorch 支持也是可选项，通常用于从其他模型格式转换。

安装 ONNX 与 PyTorch，可选
pip install onnx -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install torch torchvision -i https://pypi.tuna.tsinghua.edu.cn/simple

7. 验证 OpenVINO 与 GPU 环境

验证 OpenVINO 版本
python -c "import openvino; print(openvino.__version__)"
验证 OpenVINO 可用设备
python - <<EOF
import openvino as ov
core = ov.Core()
print("Available devices:", core.available_devices)
EOF
正常情况下应显示：Available devices: ['CPU', 'GPU']。
验证 Level Zero / GPU 设备
ls /dev/dri
clinfo | grep Device
xpu-smi discovery
uname -r

示例输出节选：

8. 下载与转换模型

安装 Hugging Face CLI：

安装 hf 命令
pip install hf

下载 Hugging Face 原始模型。设置国内镜像后下载，后续需要自行转换，模型体积较大。

下载 HF 原始模型
export HF_ENDPOINT=https://hf-mirror.com
hf download Qwen/Qwen3.6-35B-A3B --local-dir ./models/Qwen3.6-35B-A3B --resume-download

也可以直接下载适合 OpenVINO 的 int4 模型：

下载 OpenVINO int4 模型
export HF_ENDPOINT=https://hf-mirror.com
hf download OpenVINO/Qwen3.6-35B-A3B-ov --local-dir ./models/Qwen3.6-35B-A3B-ov --resume-download

如果从 ModelScope 下载：

ModelScope 下载
modelscope download --model OpenVINO/Qwen3.6-35B-A3B-ov --local_dir ./models/Qwen3.6-35B-A3B-int4-ov

如果自己下载 HF 模型后再转换，可能缺少依赖；按需安装即可。

补充转换依赖
pip install datasets -i https://pypi.tuna.tsinghua.edu.cn/simple

以 Qwen2.5-1.5B-Instruct 为例转换模型：

转换为 OpenVINO int4
optimum-cli export --model Qwen/Qwen2.5-1.5B-Instruct --weight-format int4 --task text-generation-with-past ./qwen2.5-1.5b-ov

9. OpenVINO GenAI 推理测试

模型下载完成后，可以进入 Python 交互环境测试推理。

进入 Python
python
Python 推理代码
import openvino_genai as ov_genai
pipe = ov_genai.LLMPipeline("./models/qwen3-8b-int4-ov", "GPU")
prompt = "解释一下混合专家模型的工作原理"
result = pipe.generate(prompt, max_new_tokens=1024)
print(result)
